# 🎓 Student Placement Prediction System

## 📌 Overview
The Student Placement Prediction System is a Machine Learning-based web application that predicts whether a student is likely to be placed based on academic and skill-related factors. The project also uses Google Gemini AI to generate personalized career recommendations.

## 🚀 Features
- Student placement prediction using Machine Learning
- AI-powered career recommendations using Gemini AI
- Interactive dashboard with placement statistics
- Prediction history stored in SQLite database
- PDF report generation
- Excel export of prediction history
- Responsive web interface

## 🛠️ Technologies Used
- Python
- Flask
- Machine Learning (Scikit-learn)
- Pandas

## ▶️ How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the Flask application:

```bash
python app.py
```

3. Open your browser and visit:

```
http://127.0.0.1:5000
```
